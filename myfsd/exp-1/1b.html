Understanding Hyperlinks in HTML

1. The <a> Tag

The <a> tag is used to create hyperlinks in HTML. It stands for "anchor".

2. The href Attribute

The href attribute specifies the URL of the page the link goes to.

3. The target Attribute

The target attribute specifies where to open the linked document.

  • _self (default): Opens the link in the same tab/window.
  • _blank: Opens the link in a new tab or window.
  • _parent: Opens the link in the parent frame.
  • _top: Opens the link in the full body of the window.

Summary

The basic syntax for a hyperlink is:

<a href="URL" target="target_value">Link Text</a>

Hyperlinks allow users to navigate from one webpage to another or to different sections within a page.
